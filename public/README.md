# Public Assets

## 3D Model Files
Place your 3D model and texture files here:
- `model.fbx` - Your FBX 3D model file
- `texture.jpg` - Your texture file (or .png, .jpeg)

The 3D model will be automatically loaded and displayed in the hero section.

## Image Files (Fallback)
If you prefer to use an image instead:
- `hero-image.jpg` - Hero section image

Supported formats: .jpg, .jpeg, .png, .webp
